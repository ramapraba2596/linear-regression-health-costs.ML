# 🏥 Linear Regression Health Costs Calculator

## 📌 Project Overview
This project uses a Machine Learning regression model to predict individual healthcare costs based on features like age, BMI, smoking habits, and region.

---

## 🎯 Objective
- Predict healthcare costs using regression
- Achieve Mean Absolute Error (MAE) less than 3500

---

## 📊 Dataset
Features:
- age
- sex
- bmi
- children
- smoker
- region
- charges (target)

---

## ⚙️ Technologies Used
- Python
- Pandas
- NumPy
- TensorFlow / Keras
- Scikit-learn
- Matplotlib

---

## 🔄 Data Preprocessing
- One-Hot Encoding for categorical data
- 80% Training / 20% Testing split
- Feature scaling using StandardScaler

---

## 🧠 Model
- Neural Network (Regression)
- Dense (128, ReLU)
- Dense (64, ReLU)
- Output layer (1 neuron)

Loss: MAE  
Optimizer: Adam  

---

## 📈 Performance
Model achieved **MAE < 3500**
